## Custom Domain Handle

Bluesky supports domain-based handles. If you own `wonderwomancode.com`:
1. Set handle to `@wonderwomancode.com`
2. Add DNS TXT record: `_atproto.wonderwomancode.com` -> `did=did:plc:your-did-here`
3. This verifies domain ownership and looks professional
